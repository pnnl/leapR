# leapR: Layered enrichment analysis of pathways R

leapR is a package that identifies pathways that are enriched across
diverse 'omics experiments. It leverages any tabular expression data
(proteomics, transcriptomics) using the \`SummarizedExperiment\` object.
It works with any pathway in the .gct file format.
