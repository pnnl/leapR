# KEGG, Reactome, BioCarta Pathways

KEGG, Reactome, BioCarta Pathways

## Usage

``` r
krbpaths
```

## Format

A list with 4 items

- names:

  The names of the pathways

- desc:

  Short description of the pathways

- sizes:

  Number of genes in the signaling pathways

- matrix:

  Matrix containing the genes in the pathways
